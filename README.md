# Zomato Restaurant Data Analysis 📊

Power BI and Excel analysis of restaurant data across multiple countries, uncovering
cuisine trends, delivery patterns, and pricing insights.

## 📁 Dataset
- 9,551 restaurants (~15K rows)
- Multi-country restaurant data including cuisine type, cost, ratings, votes, and delivery availability

## 🔎 Key Insights
- Most popular cuisines: North Indian, Chinese, Fast Food, and Bakery
- New Delhi has the highest number of restaurants among all cities analyzed
- Only 25.66% of restaurants offer online delivery
- Average dining cost: ₹1.2K
- Average rating: 2.67
- Average votes per restaurant: 156.91

## 🛠️ Tools & Technologies
- **Power BI** — interactive dashboard, DAX measures, and visualizations
- **Microsoft Excel** — data cleaning, transformation, and pivot table analysis
